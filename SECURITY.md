# 安全策略

## 报告安全漏洞

如果您发现任何安全漏洞，请通过以下方式报告：

1. 不要公开披露漏洞
2. 发送邮件至 dinghao12601@126.com
3. 我们将尽快回复并处理

## 安全注意事项

- 本项目使用 Nominatim API 进行地理编码，请注意其使用限制
- 所有 API 调用都经过适当的错误处理
- 建议在生产环境中使用自己的地理编码服务
- 请勿在代码中硬编码敏感信息

## 依赖安全

我们使用 uv 管理依赖，并定期更新依赖包以修复已知的安全漏洞。

## 数据安全

- 本项目不存储任何用户数据
- 所有计算都在本地完成
- 地图数据来自 OpenStreetMap
- 路线规划使用 OSRM 服务

## 最佳实践

1. 使用最新版本的依赖包
2. 定期更新系统
3. 在生产环境中使用 HTTPS
4. 设置适当的 CORS 策略
5. 限制 API 调用频率
